Containers and more Connections

A container is added, and centered a button within that container.
Using Desktop instead of App provides the GUI with a background. 
  14:app = gui.Desktop()
  15:app.connect(gui.QUIT,app.quit,None)
The container is a table 
  20:c = gui.Table(width=200,height=120)
The button CLICK event is connected to the app.close method. The button will fill the whole table cell. 
  25:e = gui.Button("Quit")
  26:e.connect(gui.CLICK,app.quit,None)
  27:c.add(e,0,0)
