tutorial on how to scroll and use a Timer object

In init(), I add bg=1 to tga_load_level so that the background layer is also loaded. 
  25:    g.tga_load_level('level.tga',1)
In init(), I also set the bounds of the level. The view will never go outside the bounds of the level. 
  31:    g.bounds = pygame.Rect(TW,TH,(len(g.tlayer[0])-2)*TW,(len(g.tlayer)-2)*TH)
In run(), I add a Timer so that I can keep a constant framerate of FPS fps. 
  41:    t = timer.Timer(FPS)
In run(), each frame I move the view to the right by SPEED pixels. 
  54:        g.view.x += SPEED
In run(), at the end of each frame, I give the timer a tick. The timer will delay the appropriate length. 
  64:        t.tick()
